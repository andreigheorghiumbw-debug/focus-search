# FocusSearch v2.6.0 🔎✨

A professional Chrome extension that restores the "Human-Only" web by purging AI-slop and low-signal content farms.

### 🎨 What's New in v2.6:
- **☕ Buy Me a Coffee Integration:** Built-in support button to help fund the official Chrome Store listing.
- **🛡️ Slop Killer:** Purge Pinterest, WikiHow, and content farms with one click.
- **🧪 AI Detox:** One-click access to Google's "Classic Web" (udm=14) mode.
- **💬 Human Mode:** Instant filters for Reddit, Forums, and Discussions.
- **💻 Dev Mode:** Fast-track to GitHub, StackOverflow, and Hacker News.

### 📁 Extension Folder
Everything you need is in the `focus_search_extension/` folder.

### 🚀 How to Install
1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer Mode** (top right)
3. Click **Load unpacked**
4. Select the `focus_search_extension` folder.
